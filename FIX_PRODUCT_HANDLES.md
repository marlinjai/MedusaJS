# Fix Product Handles - Bulk Update Script

## Problem

Products have non-SEO-friendly handles (SKUs, article numbers) instead of descriptive URLs:

- ‚ùå `/a-ct-001` ‚Üí Should be `campingtisch-100x57cm`
- ‚ùå `9105900009` ‚Üí Should be `hulse-schwingarm-gebraucht`
- ‚ùå `1591` ‚Üí Should be `anhangerbock-universal`

## Solution

Automated script to generate SEO-friendly handles from product titles.

## What the Script Does

1. ‚úÖ Fetches all products from database
2. ‚úÖ Generates SEO-friendly handles from titles
3. ‚úÖ Handles German umlauts (√§‚Üíae, √∂‚Üíoe, √º‚Üíue, √ü‚Üíss)
4. ‚úÖ Removes special characters and spaces
5. ‚úÖ Ensures uniqueness (adds `-1`, `-2` if needed)
6. ‚úÖ Skips products that already have good handles
7. ‚úÖ Shows detailed progress and summary

**Rules:**

- Lowercase only
- German umlauts converted
- Spaces ‚Üí hyphens
- Parentheses and content removed
- Special chars removed
- Max 60 characters
- No leading/trailing hyphens
